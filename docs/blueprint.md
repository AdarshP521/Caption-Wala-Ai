# **App Name**: CaptionAI

## Core Features:

- Photo Upload: Allow users to upload photos from their device.
- AI Caption Generation: Generate multiple creative caption suggestions using AI for the uploaded photo.
- Caption Selection: Enable users to scroll through and select their preferred caption from the AI-generated options.
- Caption Copy: Provide a button to copy the selected caption to the user's clipboard.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of creativity and calm.
- Background color: Very light desaturated lavender (#F5F5FF).
- Accent color: Muted rose (#D8BFD8) to complement the lavender tones.
- Body and headline font: 'Poppins' for a modern and clean aesthetic.
- Use minimalist, line-style icons in the muted rose color to represent actions and features.
- Employ a clean, card-based layout to display the photo and caption options.
- Subtle fade-in animations when loading caption suggestions.
